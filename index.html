Bienvenidos

En esta página encontrarás información sobre las distintas actividades culturales y deportivas que forman parte del CECyTEM Plantel La Paz. Nuestro objetivo es promover el desarrollo integral de los estudiantes mediante la participación en actividades que fortalecen tanto el cuerpo como la mente.

Explora cada sección para conocer más sobre las disciplinas que fomentan el trabajo en equipo, la creatividad y los valores escolares.

Propósito del Proyecto

Este sitio web fue creado como parte del examen práctico del segundo parcial de la asignatura Desarrolla software con herramientas orientadas a la productividad. Se aplicaron los conocimientos de HTML, CSS, GitHub y Jira para realizar un proyecto colaborativo con control de versiones y metodología SCRUM.

  • Organización del trabajo en Jira (modelo SCRUM)
  • Creación del repositorio en GitHub
  • Edición del código con Visual Studio Code
  • Uso de HTML y CSS para dar formato al contenido
